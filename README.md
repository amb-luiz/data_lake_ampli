# Data Lake Ampli

Projeto de arquitetura Lakehouse utilizando:

- Apache Airflow (orquestração)
- PySpark (processamento)
- Docker (infraestrutura)
- ETL pipelines

## Estrutura

- airflow-pipeline/
- pyspark-library/

## Objetivo

Centralizar e padronizar pipelines de dados, integração com ERPs e geração de bases analíticas para BI.
